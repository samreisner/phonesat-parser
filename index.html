PhoneSat Parser

Parse those packets!

This site lets you upload and parse packets received from PhoneSat satellites such as KickSat and PhoneSat itself.

This project is unaffiliated with the folks involved with PhoneSat or KickSat. It was created by using the PhoneSat packet description.

We encourage you to submit any packets you find, including the raw data, to the appropriate project team. This site is just for your convenience, so you can interpret the data you have received.
